# AlgoLearn Backend

Algoritma görselleştirme ve kod çalıştırma backend'i.

## Özellikler

- Algoritma kodlarını çalıştırma
- Adım adım görselleştirme verisi üretme
- Sorting, Graph, Searching algoritmaları desteği

## Kurulum

```bash
cd backend
npm install
npm run dev
```

## API Endpoints

### POST /api/execute
Algoritma kodunu çalıştır ve görselleştirme verisi döndür.

### POST /api/visualize/sorting
Sıralama algoritmaları için görselleştirme.

## Teknolojiler

- Node.js
- Express
- TypeScript
